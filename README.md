Tech Stack 🛠️
---
- Frontend: ReactNative, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas (NoSQL)
- Authentication: JWT (JSON Web Tokens)
- File Storage: AWS S3
- Other Tools: Axios, Multer
